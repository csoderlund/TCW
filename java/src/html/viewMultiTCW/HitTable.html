Hit Table

This table pertains to the annotation hits that are associated with each sequence.

Functions at the top

Sequences Select one or more hits followed by this function to view the sequences that contains the hit(s).

Pairwise... only one row can be selected
AA for each pair of nSeqPerform pairwise amino acid alignment for all nSeq sequences that aligned to the hit.
NT for each pair of nSeq* Perform pairwise nucleotide alignment for all nSeq sequences that aligned to the hit.
AA each nSeq to its best hitID For all nSeq sequences, align its best hit to the sequence.
AA each nSeq to the row hitID For all nSeq sequences, align the row HitID to each sequence.
* NT-mTCW only
The Pairwise options can result in many alignments, e.g. if nSeq is 20, 190 alignments will be computed; watch progress in lower left corner of viewMultiTCW.

Copy... For the selected hit, copy to clipboard a column (HitID or Hit Description).

Table...

  • Show Columns Stats: Popup a table of statistics for the numeric columns in the table. Note: if you Export the table and compute the statistics, their results will not be as precise.
     
  • Copy Table: Copies the table to the clipboard either the entire table.
     
  • Export Table: Exports the table to file.

Columns

  • Put the mouse over a column name in the Select Columns interface, and a full explanation of the column is shown in the lower left corner.
  • Columns can be added/removed using the Select Columns button below the table.
  • Columns can also be rearranged (by dragging the headers) and sorted (by clicking the headers).
  • One or more pair rows can be selected using mouse click, shift-click, or ctrl-click.
The sections are as follows, where most columns are adequately explain with the mouse-over:
General 
  nSeqThe number of sequences that have this hit.
  nBestThe number of sequences that have this as its best hit; i.e. it will be shown for the sequences "Best Hit".
  E-valuesThe best E-value of all sequences for this hit.
Hit InfoThe associated info for the hit.
