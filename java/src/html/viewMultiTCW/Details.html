Sequence Details

Details Panel

This is viewed from the Sequence table and provides information about the sequence. It contains the following two sortable tables:

Pairs: This lists all seqIDs that have a hit with the sequence, along with any shared clusters.

Hits: The top hits from each annoDB are loaded into the database when it is built, and listed here. The only columns that need explanation are the following: 

aaCov =  (Align/length of AA sequence)*100.0
hitCov = (Align/length of hit sequence)*100.0
Where align is from the Blast/Diamond hit file. The values can be >100 when there are gaps in the alignment.

The Best column has: 
EV  = best e-value hit
AN  = best annotation hit
WG  = best hit with GOs
ALL = EV+AN+WG

Frame Panel

This replaces the current sequence panel with a display of the sequence; if it is a nucleotide sequence, then it shows information about the frame and best hit in relation to the sequence.

GO Panel

This will only be available if GOs were added to the database when it was built with runMultiTCW.

There are four options, where selecting one will replace the current sequence panel with a GO panel.
Assigned GOs for hitsGOs that have been directly assigned to at least one hit.
All GOs for hitsBoth assigned and inherited GOs for all hits.
Assigned GOs for the selected hit (or best)All assigned GOs for the selected hit. If no row in the Hits table is selected, then the best hit is used.
All GOs for the selected hit (or best) All assigned and inherited GOs for the selected hits. If no row in the Hits table is selected, then the best hit is used.

Pairwise...

The first 4 options apply to the pair table. The last option applies to the hit table. In both cases, (1) if no row is selected, the first row will be used, (2) only one row can be selected.
AA to pair seqIDPairwise alignment of amino acid sequence from the selected pair sequence.
NT to pair seqID* Pairwise alignment of the nucleotide sequence to the selected pair sequence.
AA,NT,CDS to pair seqID* Align the selected pair using the amino acid sequence, CDS sequence, and nucleotide sequences. The resulting panel will have these 3 alignments.
5UTR, CDS, 3UTR to pair seqID* Align these three regions separately for the selected pair seq. The resulting panel will have these 3 alignments.
AA to hitID The selected hit is aligned to the sequence.
* NT-mTCW only

Copy...

  1. Detailed sequence: The first 3 options are for this.
  2. Pairs: The next 4 options apply to the selection in the sequence Pairs Table; if no row is selected, the first row will be used.
  3. Hits: The remaining options apply to the selection in the sequence Hits Table; if no row is selected, the Best Hit will be used.
If there are AA-sTCWdbs in the mTCW, the Copy NT sequence and Copy CDS sequence options will not be available.
