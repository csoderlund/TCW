Sequence Details

This is viewed from the Sequence table and provides information about the sequence. It contains two sortable tables.

Pairs

This lists all seqIDs that have a hit with the sequence, along with any shared clusters.

Hits

The top first hits from each type are loaded into the database when it is built, and listed here. The only columns that need explanation are the following: 
aaCov =  (Align/length of AA sequence)*100.0
hitCov = (Align/length of hit sequence)*100.0
The values can be >100 when there are gaps in the alignment.

The Best column has: 
EV  = best e-value hit
AN  = best annotation hit
WG  = best hit with GOs
ALL = EV+AN+WG
