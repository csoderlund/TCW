viewMultiTCW

Instructions

The multiTCW database is created from two or more singleTCW databases (datasets). A dataset may have both nucleotide (NT) sequences and amino acid (AA) sequences, or only AA sequences. If the database contains both NT and AA sequences, then the AA sequences are the translated ORFs of the NT sequences.

Clusters: There can be one or more clusters of type BBH (bi-directional best hit), Closure, BestHit (best annotation), orthoMCL or user-defined.

Pairs: There is a pair for two sequences if they have a AA or NT hit, where the hits are computed when the database is created by searching a file of all sequences in the database against itself using a heuristic search program (Diamond or Blast). A NT-only databases will have additional information for NT pairs in a cluster, such as number of synonymous, nonsynonymous codons.

A few general rules:
  1. Filters: Click a (Cluster, Pair, Sequence, Hit) filter on the left panel. Select your filters and click the "View Filtered" button, which results in a table on the right and its label listed on the left. You can always go back to the table by clicking its label.

  2. Samples: All tables (Cluster, Pair, Sequence, Hit) can be quite large, hence, the samples are a quick way to see the information provided in a table.

  3. Clicking on a function in the table often results in another label on the left. Functions that start with "Show" result in popup windows with the information.

  4. All panels have a Help to describe the options.

  5. All tables have a Select Columns that allows adding or removing columns from the table. Putting the mouse over a column name in the panel shows its description in the lower left corner.

  6. Results:
    • Results on the left can be removed by clicking the 'x' at the end.
    • List Results: Lists of all results under any of the ">" sections. They can all be removed together by clicking the "Remove All", or remove only the selected results.
