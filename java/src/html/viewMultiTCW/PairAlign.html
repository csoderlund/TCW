Multi-line Text Alignment

This refers to the multiple pairwise Align... and the three-way "Align:" AA..., CDS..., NT..., 5UTR..., 3UTR... options. Selecting a '...' button results in a pop-up menu with the following options (depending on the view). Once Ok is selected from the pop-up, a multi-line textual alignment will be shown; there is a legend at the bottom of the window

All pop-ups have the following option:
Remove hanging sequence from ends If checked, overhangs are removed, where an overhang is an extension of one sequence that the other sequence does not have. If uncheck, the full alignment is shown.
DisplayThe number of characters per row.

AA Align... and AA...

Display: 60, 90 or 20 -- is the number of amino acids in a line.

NT Align... and NT...

Display: 60 or 90 -- is the number of bases in a line.

Annotate: Has the following options:
MatchShows a symbol for the mismatched nucleotides.
ts/tvShows ts (transitive) and tv (transversion) substitutions.
CpGShows all CpG dinucleotides.

CDS...

Display: 60 or 90 -- is the number of bases in a line.

Annotate: With the following options:
MatchShows synonymous and non-synonmous mismatches.
Amino AcidShows the corresponding AA character for each codon.
DegenerateShows fourfold (4d) and twofold (2d) degenerate codons. Degenerate codons code for the same amino acid, but 4d can have any of the 4 bases in the ith position and 2d can have 2 or 3 alternative bases in the same position.
ts/tvShows ts (transitive) and tv (transversion) substitutions.
CpGShows the location of CpG dinucleotides within codons (but not CpG dinucleotides that span codons).

5'UTR... and 3'UTR...

Display: 60 or 90 -- is the number of bases in a line.
