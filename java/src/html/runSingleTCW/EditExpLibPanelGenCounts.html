Build combined counts file

Often the counts for each replicate are provided in separate files, which need to be combined into one table for import into TCW. This interface is designed to make that easier.

To use this interface, your count files need to have a two-column format without headers: 

Sequence_name1	count1
Sequence_name2	count2
...

Add Directory of FilesGo to the directory with your count files and select Open. All files will be checked for the correct format, and the correct ones will be entered into the table. The "Rep name" is extracted from the file name; it can be changed by editing the Rep Name column.
Add Rep FileA file chooser window will pop up to chose a file. The "Rep name" will be determined from the file name; it can be changed by editing the Rep Name column.
Remove RepSelect a row to remove it from the table.
Clear Clears the table contents.
Generate FileThe file Combined_read_counts.csv will be generated, which will have one column for each of your original files, with the specified Rep names as headers. This will be entered as the Count File in the Add or Edit panel, and when you press Keep on that panel, the information will be transferred to the main panel.
Close The panel will be closed without discarding the table contents. However, if you exit runSingleTCW, the information will be lost.

To edit the Rep Name column, you may have to click at the very beginning or end of the cell you want to change; the cell will turn white indicating you can edit.
