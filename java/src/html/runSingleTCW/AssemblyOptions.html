Assembly Options

The default assembly parameters have been tuned on a large variety of datasets. See online documentation for more detailed information on the assembly process. Note that TCW can assemble Sanger and 454 raw reads, as well as existing transcript sets, but cannot assemble raw Illumina reads or similar datasets.
Assemble
Clique Determines which overlaps are accepted from the clique MegaBLAST output.
Minimal overlap: minimum matching bases allowed.
Minimal %identity: minimum identity of the match.
Maximal overhang: maximum number of non-matching bases attached to the putative joining region.
Clique Blast Eval E-value for MegaBLAST used to form cliques.
Clique Blast Parameters Additional parameters for MegaBLAST used to form cliques.
Self Join Determines which MegaBLAST results are accepted for overlapping 5' and 3' mate pairs. See description under "Clique" above.
Bury Blast Eval E-value for the initial bury blast.
Bury Blast Identity %Identity (-p) for the initial bury blast.
Bury Blast Params Additional parameters for the initial bury blast.
CAP Args Parameters for running cap3 (also used for assembling cliques).
TC Blast Eval E-value for blast overlaps during contig merge stages.
TC Blast Params Additional parameters for blast overlaps during contig merge stages.
TC#N Table The table defines N rounds of blasting contigs and merging. The parameters are for loading the Nth MegaBLAST results, and are the same as described for "Clique" above.
The last three cells can be edited; click in cell, hit backspace, and the cell will turn white indicting it can be edited.
