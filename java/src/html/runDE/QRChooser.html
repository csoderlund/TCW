Database Chooser for DE

Select a sTCWdb (singleTCW database), then do one of the following:
  1. Double click to bring up the DE window with the sTCWdb.
  2. Select Launch to bring up the DE window for the sTCWdb.
  3. Select Overview to pop up a window showing the overview of the sTCWdb.
The DE window has Help for computing DE. Multiple DE window can be active, but only one can run R at a time.

Close and Exit

  1. Close: Close the DB Chooser window only (do not close any exiting DE windows).
  2. Exit: Close all windows and exit the R environment saving the session.
If at any time you respond to the R prompt: 
Save workspace image? [y/n/c]:
all windows will close.
