Strong suggestions:

  • Only download what you need!
  • Avoid the TrEMBL Bacteria and Full unless you really need them. Using a fast university connection, it took 3 hours to download TrEMBL Bacteria and an hour for TCW to parse it.
  • Do not perform constant downloads, it is a drain on the UniProt servers.
I haven't tried the Full TrEMBL in a few years, as I find it adequate to download the SwissProt and TrEMBL relevant taxonomic databases, then the SwissProt Full database.
