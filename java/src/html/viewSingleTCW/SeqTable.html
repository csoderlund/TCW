Sequence Result Table

Shows search results from "Filters" or "Show All".

Columns

Add/remove columns

Change columns as follows:
  • Select "Columns" link on the left panel.
  • Return to the desired table (selecting its tab on the left).
  • Selecting "Refresh Columns" will redraw the table with the altered column selection.

Sorting

Columns may be sorted ascending or descending. Click on the header once for descending, then again for ascending; or, right-click for ascending. Note: The DE columns have negative and positive values, where negative means the first condition has TPM (or RPKM) less than the second. The sort on the DE column ignores the sign so the right click sorts all DE to the top.

Column re-order

Columns may dragged to different positions, by clicking and dragging the headers.

Table...

Show Column Stats

Shows statistics for all numeric columns in the table.

Copy

Copies the table to the clipboard, which can then be pasted into a file.

Export

The table data can be exported to a file with 6 different options:
  • Table of columns: tab-delimited copy of the table.
     
  • Sequences from table: fasta file of sequences for the seqIDs in the table.
     
  • Translated ORFs: fasta file of translated ORFs for the seqIDs in the table, where the translated ORFs correspond to the TCW selected frame and coordinates.
     
  • Library replicates from table: combined file of library replicates for the seqIDs in the table
     
  • Hit Sequences from table: creates a file of the combined best hit sequences that are either:
    1. Best Eval: the best E-value per sequence.
    2. Best Anno: the best annotation per sequence.
    3. Both: all Best Eval and Best Anno hits.

  • Sequences with Best GO: creates a text file of all sequences that have a Best GO. The output corresponds with the "WG GO" column, which is the best hit for the sequence with GOs assigned.
     
  • GOs from table: list GOs associated with the seqIDs in the table along with the count (#Seqs) of the number of sequences that have the GO (assigned or inherited). The output columns are: GO term, #Seqs, Level, Domain, Description and all p-values.
    1. GO Level:
      1. Level 1 is the three domains. Level 2 are the GO terms directly beneath a domain, etc (see Basic GO Annotation 'GO Help' for more explanation).
      2. If you leave this blank, it will output all GOs.
      3. If you just want a couple of levels, e.g. 2 and 3, first export level 2, then export level 3 to the same file and use the append option (there will be a pop-up asking whether you want to append).
    2. Any domain: This toggles through the three main domains. If you select one, it will only export the GOs in that domain.
    3. E-value: If you leave this blank, it will ignore the E-value. Otherwise, this applies to each sequence-hit E-value where the hit has the GO. The #Seqs column is the number of sequences that have at least one hit with the GO and its E-value< the value.
    See GO Term for more Export options.
