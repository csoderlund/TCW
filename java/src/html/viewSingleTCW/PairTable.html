Sequence Pairs

The table shows similar-sequence pairs calculated during the annotation stage (see below).

Select Pairs Columns on the left panel to add/remove columns from the table. After choosing columns, return to the table and use Refresh Columns.

Columns may be sorted ascending or descending by clicking the headers, and may be rearranged by dragging the headers.

Align selected pair

This aligns the two sequences using a dynamic programming algorithm. The resulting panel is a graphical display of the aligned sequences in the best frame, and the panel has the following additional alignment options:

Align in best frame (default) It aligns the nucleotide using the pre-computed orientation and the amino acids using the pre-computed frame.
Align in all framesAlign the nucleotide sequences in both orientations. Align the amino acid with the precomputed frame. Then compute the remaining 35 possibilities.
Align the top 3 shared hits The 3 best hits shared between the two sequences will be aligned to each sequence, for a total of 6 alignments.

View sequences

This creates a sequence table of the two sequences. The resulting table is placed under Filters (i.e. TCW treats it as a filtered table).
