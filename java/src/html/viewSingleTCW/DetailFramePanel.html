Sequence Frame

If this is a protein database, the sequence is displayed and the rest of this is irreverent.

During annotation, TCW selects the frame that is most likely to be coding. It uses a combination of protein hit, 5th-order Markov Chain and length (for more detail, see www.agcol.arizona.edu/software/tcw/doc/stcw/ORF.html).

Frame Options - second row of buttons

Frame By default, the TCW selected frame is shown. Select a different frame to view the its best frame.
ORFs/NT Changes the display from ORFs/NT to Scores/AA. The ORFs/NT shows the nucleotide codons and the Scores/AA shows the amino acid sequence. The top display is discussed below.
Start By default, start sites are highlighted in grey (alternative start sites are lower case, ATG is upper case). This toggle changes the highlight from grey to cyan.
CDS Highlights the entire selected ORF in green.
Hit Highlight the hit region using italics; the drop-down beside this can change the italics to blue. This is only present if the "Best Bits" hit is to a protein and to this frame. The hit coordinates are from the heuristic search program (e.g. Blast) and assigned during annotation.

Top text display

ORFs/NT The first line provides number of the selected RF along with any remark assigned by the ORF finder; see the Basic Sequence Help for a description. The "Seq Length" is the full length of the nucleotide sequence.

The following lines have information about the best ORF in each of the 6 frames, where the ORF being viewed will have a ">" in front of it. The information is:

  • The ORF coordinates are listed, where a single quote (') after the start codon indicates that the ORF does not have a start codon, likewise for the stop codon.
     
  • The 5th-order Markov model and Codon Usage scores.
    • A good Markov score is positive and the best score for all frames of the given ORF (see the Scores/AA display); if a score is not good, it will have a single quote (') beside it.
    • The best Markov score is the best over all 6 ORFs shown in the ORFs/NT display. If the selected best ORF is not the best, the sequence will be remarked with "ORF!Mk".
  • The number of stop codons in the entire sequence. If there are stop codons in the hit region, then the sequence will be remarked with "StopHit".

Scores/AA The 6-frame score for ORF frame N, where N is the frame being displayed. It shows the Markov and Codon scores for the current ORF sequence in all 6 frames, with the starting location of the ORF equal to position 1,2,3,-1,-2,-3. If the Markov score on the ORFs/NT display has a single quote beside it, then the first score will either be negative and/or not the best of all frames for this ORF.

Sequence Display Coordinates

In the lower panel, the start and stop of the ORF are highlighted in green. To calculate the nucleotide coordinate of a particular base:
  • Forward:
        Y-axis# + X-axis# = last base of the codon.
        Y-axis# + X-axis# - 2 = first base of the codon.
  • Reverse:
        Y-axis# - X-axis# = last base of the codon.
        Y-axis# - X-axis# + 2 = first base of the codon.
  • Hit coordinates:
    • Forward: (Y-axis# + X-axis# - 2) to (Y-axis# + X-axis#)
    • Reverse: (Y-axis# - X-axis# + 2) to (Y-axis# - X-axis#)

To calculate the coordinate of a particular amino acid:

  • Forward: Y-axis# + X-axis#
  • Reverse :Y-axis# - X-axis#
