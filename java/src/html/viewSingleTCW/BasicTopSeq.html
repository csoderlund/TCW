Top row of options for Basic Sequence

Seq Table  Select one or more seqIDs in the table, then select this button. The sequences will be shown in the main sequence table.

Seq Detail  Select one seqIDs in the table, then select this button. The sequences will be shown in the Sequence Details display.

Copy...  Select one sequence to copy to clipboard the SeqID, Best HitID, or Orig ID.

Table...

  • Copy table The table will be copied so that you can paste it into a file.
  • Export A window will pop-up with a file chooser, and the table will be written to the selected file.
