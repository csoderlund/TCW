Pairs Columns

Use this interface to select columns for the results tables from pair queries. Note that sections are collapsible; to expand a section and see the column options, click on the "+".

The description of each column is explained on the column page.

Sequence Information

Pair#The pair number.
Seq ID 1The Seq ID of the first sequence of the pair.
Seq ID 2The Seq ID of the second sequence of the pair.
Shared HitThe best shared hit between the pair. This is blank if they do not share a hit.

Hit results for the pair

These four columns are from searching all sequences against each other using one or more of the following:
tblastx(AA)   6-frame translated of nucleotide sequences
blastp(AA)   Either translated ORFs (NT-sTCW) or proteins sequences (AA-sTCW)
blastn(NT)   Nucleotide sequence comparison.

Columns:

Hit E-valueThe E-value of the first hit for a pair.
Hit %SimThe %Similarity of the first hit for a pair.
Hit AlignThe alignment length of the first hit for a pair.
Hit Type
AAThe pair had a tblastx hit.
ORFThe pair had a blastp hit.
NTThe pair had a blastn hit.
DPThe sequences ORF frames do not match with the first hit frames.
DP example: AA::NT::DP (1,2)->(2,2)
AA::NT:The pair has an AA and NT hit, where the Hit values (E-value, %Sim, Align) are from the AA hit.
DP:The first sequence's ORF is in Frame 1, yet its AA hit with the second sequence was in Frame 2.

Total Coverage

The "Total Coverage" columns are based on a semi-global dynamic programming algorithm.

Columns:

NT %SimThe number of match nucleotides divided by the alignment length.
NT LenThe alignment length, which does not include overhangs at the beginning or end.
NT %LenThe number of match nucleotides divided by the longest length. If this is 100, then the sequence align their entire length. If "NT %Sim" is also 100, the pair of sequences are the same.
AA %SimThe number of match amino acids divided by the alignment length; both exact and amino acids substitutions (blosum score>0) are considered a match; two aligned stop codons are not counted as a match.
AA LenThe alignment length, which does not include overhangs at the beginning or end.
AA %LenThe number of match amino acids divided by the longest length. If this is 100, then the sequence align their entire length.
