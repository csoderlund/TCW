Basic Queries for Sequence

This provides some basic sequence searches; for more involved searches, use the Filter bullet on the left panel. This allows you to "build" a table of results from multiple queries, which can be copied to Clipboard, exported, or view the individual sequences with View Selected Sequence.

Search

Choose one of four search modes and enter a string to search on.

  • Seq ID: Sequence name
  • Orig ID or Longest or None: (1) Original name if Seq ID != Orig ID (2) Longest read of contig if assembled, (3) no query if neither of the previous.
  • TCW: TCW Remarks are entered when the database is built; they provide information on the ORFs, and if the sequence has hits in multiple frames.
  • User: User Remarks may be associated with each sequence (entered with runSingleTCW).
Search rules
  • It first tries to find an exact match. If found, the results are listed. If not found, it then tries to match the substring anywhere in the hit ID or description.
  • To force it to match at the beginning of the string, enter a '%' at the end of the substring, e.g. 'mitochondrial%' to match everything that starts with 'mitochondrial'. NOTE: since '%' is a special character, it cannot be in a search string as a literal.
Load File: The file should contain one Sequence ID per line (make sure Seq IDs is selected); likewise for Orig ID, TCW (remark) and User (remark). This performs an exact search.

Results

Select one or more filters, then click BUILD TABLE or ADD to TABLE to execute the search, which builds a new table or adds to the existing table, respectively.
Columns can be added/removed (Columns), rearranged by dragging and sorted by clicking on a column heading.

Modify Table

  • Delete Selected and Deleted Unselected: The selected rows or unselected rows will be removed, respectively. The corresponding rows will be removed from the other (not-displayed) list.
  • Select All and Unselect All: Selects and unselects all rows, respectively.

Columns

TCW RemarkAssigned by TCW during runSingleTCW ORF computation
User RemarkUploaded by the user with runSingleTCW.
CountTotal count from all conditions/libraries.
Orig ID or LongestSee Search above.
Best HitIDThe HitID of the hit with the best e-value.

Top row of options

View Sequence: Select a one or more seqIDs in the table, then select View Sequences. The sequences will be shown in the main table, where you can select it to view the Sequence Details.

Copy...: Select one sequence to copy to clipboard the SeqID, Best HitID, or Longest.

Table...:

  • Copy table: The table will be copied so that you can paste it into a file.
  • Export: A window will pop-up with a file chooser, and the table will be written to the selected file.
