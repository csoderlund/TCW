Sequence Details

Best ORF: Shows the RF (reading frame) and coordinates; a "'" indicates a missing start or stop. The TCW remark is at the end (See Basic Sequence Help for the TCW Remark Legend).

DB hits:

  • The columns E-value, %Sim, Align, Start and End are from the best search results for the hit (i.e. from BLAST or DIAMOND).
  • The column Align is in the hit coordinates (e.g. amino acids if the hit is a protein), whereas the Start and End are in the sequence coordinates.
  • The options GO..., Align... and Show/Copy/Export... all have options that use the DB hits table contents or selection.

Detail Options - second row of buttons

  1. Counts toggle: Toggles between counts, TPM (or RPKM), and Replicates (if there are replicates).
  2. DB hits toggle: Changes the set of hits listed in the DB hits table.
    • Best Hits (BS, AN, GO). See Introduction for more information on the following.
      • BS is best bit-score.
      • AN is best annotation.
      • WG is best with GO.
      • The Best BS, AN, and WG may all be the same, or any two of them the same.
    • Best per annoDB: In preprocessing, the sequences in the database are generally searched against one or more annotation databases, which are referred to as an annoDBs. For example, the sequences may have been searched against SwissProt Plants and TrEMBL Plants taxonomic databases. The best hit for each is listed.
    • Distinct regions: A set of hits that represent the regions of the sequence for which hits align to based on hit coordinates. Note the Start and End coordinates in the table, they will not be the same, though may overlap some.
    • Unique annotation: Determines the unique descriptions, after filtering on: (1) ignore putative, hypothetical and predicted, (2) remove last character of description as often a number or letter.
    • Unique species: Lists one of each species that was hit
    • Total: All hits to the sequence.
    Note: sometimes its necessary to click in the window to get it to change the table, after changing the DB Hits option.
  3. Show... This only has one option, which is Selected Hit Info. This shows all information associated with the hit to sequence.
  4. Copy... This is a menu of options to copy content to the clipboard (to "Paste" elsewhere). The first five refer to the sequence. The last three require a hit to be selected in the DB hits table.
  5. Export... Writes a file of the sequence or DB hits.
