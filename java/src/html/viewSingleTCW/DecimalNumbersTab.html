Decimal Numbers

There are two display options, both use the following two variables:
  1. Decimal places: This is used for numbers >1. It is the number of digits after the decimal point, e.g. 123.45 has two decimal places. Minimum: 0, Default: 2.
  2. Significant figures: This is used for numbers <1 and E-notation displays. This is the number of important digits (see Wikipedia for a good description), e.g. 0.012 and 1.2E-2 both have two significant figures. Minimum: 1, Default: 2.
Display options:
  1. E-notation: This allows the user to set how big or small the absolute value of the number can be before switching to E-notation. Note, E-notation is scientific notation where 'E' is used to represent the exponent. To view all numbers without using E-notation, enter 1E+20 and 0 in the two boxes, respectively. To view all numbers in E-notation, enter 1 in both boxes.
  2. Java defaults: Java determines when to switch to E-notation.
This is used for both the Sequence table display and the Basic queries. A few quirkiness:
  1. Sequence table: When you change any of the values, the Sequence table will immediately be changed, i.e. you do not need to redisplay the columns.
  2. Basic tables: When you change any of the values, the Basic table will NOT immediately change, you must redisplay the table.
