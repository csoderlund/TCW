Results

Sequence Results

All labels on the left panel will be shown in this table.
  • These are either table or sequence detail labels.
  • A label representing a table will include a ":", e.g. "Filter1:".
  • Removing a table label will remove any sequence detail labels listed for it; the Summary contains the table label up to the ":" and the sequence detail label listed under the table on the left panel.
  • The All Seqs, Find Hits and Sequences labels do not have entries in the table, so any sequence detail label needs to be remove individually or with the Remove All Results.
Remove All Results: All labels will be removed and the table cleared.

Remove Selected Result: Select a result followed by this option to remove it, or the label can be removed by selecting the 'x' by its name on the left.

Double-click a row will show its corresponding table or sequence detail.

Pair Results

Only filtered tables are listed, which have a corresponding label on the left under Pair Filters.
  • Removing a filtered table will remove all the alignments shown on the left beneath the filtered label.
  • If the Seq Table is selected for one or more pairs, the resulting sequence table will be listed under Filters and be listed on this Results page.
Remove All Results: All Pair labels will be removed and the table cleared.

Remove Selected Result: Select a label followed by this option to remove it, or the label can be removed by selecting the 'x' by its name on the left.

Double-click a result to show its corresponding table.
