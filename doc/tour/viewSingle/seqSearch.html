Sequence Search
The Filters provide various queries on the sequences, whereas the Sequence option under Basic Query provides search by sequence ID or by remark. The user can add remarks through the runSingleTCW interface. TCW adds remarks during frame computation; one such remark ("Multi-frame") indicates that the sequence has aligned hits in multiple frames.

One or more sequence can be selected followed by View Sequences and they will be shown

A sequence can be selected to see its Sequence Details. If multiple sequences in the Sequence Table.
