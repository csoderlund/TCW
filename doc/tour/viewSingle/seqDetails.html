The Sequence Detail view can be displayed from the Main Table by selecting a sequence followed by View Selected Sequence.

Details

The image is an example of the detail page.

Second row of options:

  • The Counts pull-down can change the Counts: to RPKM or Replicates.
  • The All hits can be changed to display only Best Hit, Best per AnnoDB, Unique Species, Distinct Regions.
  • The Copy/Export options are shown on the right.

Frame

Go to top
The Start and Stop are highlighted in Green. The location of the best hit is shown in italics, where it forms a perfect ORF in this example (Stop is not shown). The sequence can be shown in alternative frames using the Frame pull-down.

The TCW selected best frame is indicated by the addional information shown on the right (beside Stop). The TCW algorithm uses hit information, 5th-order Markov score, and start/stop codons to compute the best frame.

Align

Go to top

Dynamic programming is used to compute the alignment.

The green is gaps, the pink is non-synonymous mismatches and the purple is synonymous mismatches.

Selecting a hit followed by "Alignment" shows a multi-line sequence view.

There are options to show the alignment in all frames and/or to show alignments to multiple hits.


GO

Go to top
The image on the right shows the union of the GO terms for sequence NNU_025907.

Most of the columns are obvious, except the following two:
➤Level: A GO term can be at multiple levels in the tree. This number is its maximum level.
➤Evidence: This shows evidence code for the protein being assigned the GO term. The number before an Evidence code is the number of hits that were assigned to the GO with the corresponding evidence.

Selecting a GO term followed by Hits provides details of the hits that are assigned the GO.
Selecting a GO term followed by Ancestor provides the list of ancestors for the GO.
