GO Annotations

The contents of this panel will depend on how the database was built with runSingleTCW:

  • This panel will exist only if UniProt databases were used in annotation.
  • The Slim option will exist only if a GO_slim set was selected (not available in this sTCWdb).
  • The DE: row of options will exist only if GOseq has been been run (see runDE).

In the following image, the GOs that are up-regulated for Root compared to Stem and Leaf are shown. The #Seqs are the counts for up-regulated sequences for the respective GO.

GO Terms may be assigned to a hit ID, or it may inherit a hit ID; see GO explanation for details.
