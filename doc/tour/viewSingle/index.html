viewSingleTCW provides query and display of the sTCW database built with runSingleTCW. It allows the user to explore all aspects of their data and annotation.

General
➤Overview: Overview is a file of the complete overview, where the first part is shown on the right.

Reproduce are instructions on how obtain the table of results corresponding to the overview statistics.

➤Find Hits: Search an user supplied sequence against the sequences in the database or an external database.

Sequences
The items under this label produces the Sequence Table. A row can be selected to view the Sequence Details.

Basic Queries
➤Sequences: Search by seqID or remark.

➤AnnoDB Hits: Search by hitID, taxonomy, e-value, etc.

➤GO Annotation: Search by GO term, term_type, etc.

The initial viewSingleTCW display.

Similar Pairs: The items under this label produces a Pairs table of similar sequences.
