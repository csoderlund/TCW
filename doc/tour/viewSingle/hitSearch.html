Hit Search

The annoDB Hits option under Basic Query searches by hit ID, hit Description and/or filters.

  • Select the desired filters followed by BUILD TABLE.
  • The line above the table starting with "Results" show the filters applied.
  • Select a hit followed by View Selected Hits will show all the associated sequences in the Main Table.
  • Group by Hit ID -- unchecked changes the table to show the hits with sequence IDs.

    Uncheck the Group by Hit ID to view the sequences.

    The align option shows the alignment of all sequences that have been assigned to a hit (based on Blast values during the building of the database). The alignment shown below uses dynamic programming.

    Search and Filter

    Search: This rows allows searching for a specified hit ID or description, with or without the rest of the filters applied.

    Filters Seq: This row of options apply to the sequences that align to the annoDB hit (generally protein, but can be nucleotide).

    • Selecting Exp brings up the window on the right. By selecting "RTZ_Ro", which is the p-value for the counts of RhiTiZo (rhizome) compared to Root, the result would be all annoDB hits that have at least one sequence with RhTZ_Ro<0.05.
    • The Filters Hit would also be applied, e.g. if E-value is selected and set to 1E-100, the sequence-hit pair would also require an E-value<1E-100.
    Filters Hit: This row of options apply to the annotation assigned to the annoDB hit.
    • Selecting AnnoDBs brings up a window of all available annoDBs, e.g. "sp -- plants", where one or more of these may be selected. By default, they are all selected.
    • Selecting Species brings up a window of all available species (shown on the right), where one or more may be selected. By default, they are all selected.

    The window on the right are the available columns for this table. These can be changed without running BUILD TABLE again.
