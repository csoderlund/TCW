GO Search

The contents of this panel will depend on how the database was built with runSingleTCW: (1) This panel will exist only if UniProt databases were used in annotation. (2) The Slim option will exist only if a GO_slim set was selected. (3) The DE row of options will exist only if GOseq has been been run (see runDE). Plus, the DE-trim option will exist only if there are DE values; the trimming algorithm removes redundancy, as described in Soderlund et al. (2014) PLoS ONE.

In the following image, the GOs that are DE for RZT_Ol, RZT_Ro and RTZ_St are shown. The #Seqs are the counts for up-regulated sequences for the respective GO.

GO Terms may be assigned to a hit ID, or it may inherit a hit ID; see GO explanation for details.
