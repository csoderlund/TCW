Pairwise

Go to top
This drop-down has the following options, where all options use dynamic programming for the selected sequence(s):

  1. AA for each pair
  2. NT for each pair
  3. AA, CDS and NT alignments for one pair
  4. 5UTR, CDS and 3UTR alignments for one pair
  5. AA, CDS and NT alignments for one pair
  6. AA to sequence best hit
  7. AA to cluster best hit (Clusters only)

AA, CDS and NT alignments of a pair. The text alignment can be viewed by clicking the AA, CDS or NT buttons.


Menu for the CDS text alignment: Text alignment of the CDS showing matches. The '|' are mutations in synonymous codons, the '*' in nonsynonymous codons.

MSAdb

Go to top
This option is only available for Sequence view for clusters, as the multiple sequence alignment (MSA) is computed when the database is built.

MSA...

Go to top
This drop-down has the following options, where all options execute the MSA program for the set of selected sequences:

  • MUSCLE-AA: Align the AA sequences.
  • MAFFTA-AA: Align the AA sequences.
  • MAFFTA-CDS: Align the CDS sequences.
  • MAFFTA-NT: Align the NT sequences (i.e. full transcript).
The display is the same as shown above for MSAdb.
